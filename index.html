Bem vindo ao Arduino!

Arduino é uma plataforma programável de prototipagem eletrônica de placa única e hardware livre, que permite aos usuários criar objetos eletrônicos interativos e independentes, usando o microcontrolador Atmel AVR ou ARM com suporte de entrada/saída embutido, uma linguagem de programação padrão, essencialmente C/C++.

Você sabe o que é Arduino? Há diversas configurações e versões de placas diferentes no mercado, cada uma com seus recursos e recomendações. Mas antes mesmo de pensar em começar a se aventurar na computação física, robótica e afins, você precisa saber o que é um Arduino, e se perguntar para quê irá usá-lo. Para isso, vou explicar um pouco mais sobre o que é Arduino. Venha conferir.

Projetos feitos com Arduino

Rastreador Solar

Este é um projeto de rastreador solar de dois eixos com Arduino Uno. Ele usa sensores de resistor dependente de luz (LDR) para controle automático ou um potenciômetro para controle manual. O projeto é de código aberto e utiliza instrumentação virtual baseada em Excel para gravar e apresentar dados. Ele foi projetado para ser barato, compacto e versátil, ajudando os alunos a entenderem a teoria de controle e sua aplicação.

Sistema de Segurança Ultrassônico

O Ultrasonic Security System é um projeto de sistema de segurança baseado em sensor ultrassônico, que detecta a presença de objetos em seu alcance e aciona um alarme sonoro. É uma solução simples e eficaz para monitorar ambientes e proteger contra intrusos.

Drone Autopiloto

Um drone com piloto automático baseado em Arduino Uno, equipado com vários sensores e uma câmera sem fio controlada por dois microcontroladores.

Brinque com o Arduino!

Passo 1: Configuração do Hardware Conecte o Arduino ao computador usando um cabo USB. Conecte um LED ao pino 13 do Arduino. Conecte o terminal positivo (ânodo) do LED ao pino 13 e o terminal negativo (cátodo) a um resistor de 220 ohms. Conecte o outro terminal do resistor (ou seja, o lado sem o LED) ao GND (terra) do Arduino. Passo 2: Programação do Arduino Agora, vamos escrever um código simples para fazer o LED piscar no Arduino. Abra a IDE do Arduino no seu computador. Crie um novo projeto e cole o seguinte código: 
void setup() {
  pinMode(13, OUTPUT); // Configura o pino 13 como saída
}

void loop() {
  digitalWrite(13, HIGH); // Liga o LED
  delay(1000); // Aguarda 1 segundo
  digitalWrite(13, LOW); // Desliga o LED
  delay(1000); // Aguarda 1 segundo
}
Para controlar o Arduino a partir do site HTML, precisamos configurar um servidor no Arduino para lidar com as solicitações do site e controlar o LED. Volte para a IDE do Arduino. Crie um novo projeto e cole o seguinte código:
#include <Ethernet.h> // Inclua a biblioteca Ethernet se estiver usando um shield Ethernet

byte mac[] = {0xDE, 0xAD, 0xBE, 0xEF, 0xFE, 0xED}; // Endereço MAC do Arduino
IPAddress ip(192, 168, 0, 10); // Endereço IP do Arduino

EthernetServer server(80); // Cria um servidor na porta 80

void setup() {
  Ethernet.begin(mac, ip); // Inicializa a conexão Ethernet
  server.begin(); // Inicia o servidor
  pinMode(13, OUTPUT); // Configura o pino 13 como saída
}

void loop() {
  EthernetClient client = server.available(); // Verifica se há clientes conectados

  if (client) {
    while (client.connected()) {
      if (client.available()) {
        char c = client.read(); // Lê o caractere enviado pelo cliente

        if (c == 'ligar') {
          digitalWrite(13, HIGH); // Liga o LED
        } else if (c == 'desligar') {
          digitalWrite(13, LOW); // Desliga o LED
        }

        client.println("HTTP/1.1 200 OK"); // Envia uma resposta de status HTTP
        client.println("Content-type:text/html");
        client.println();
        client.println("<html><body>Comando recebido.</body></html>"); // Envia uma resposta HTML
        break;
      }
    }
    delay(1); // Pequena pausa antes de enviar a resposta
    client.stop(); // Fecha a conexão com o cliente
  }
}
